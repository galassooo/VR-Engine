Detailed Description

Represents the material properties of a 3D object, including color, shininess, and optional textures.

The Material class encapsulates visual properties used in rendering, such as color, shininess for specular highlights, and an optional diffuse texture.

Applies this material's properties to the current shader.

Uploads ambient, diffuse, specular, shininess, and emission parameters to the ShaderManager, binds the diffuse texture if any, and configures OpenGL blending based on alpha transparency.
