Detailed Description

Manages Shaders to be built into the GPU Program for the graphics engine.

The ShaderManager implements the Singleton pattern to provide centralized management of all shaders and programs. It handles:

  • Registration and binding of default shaders
  • Management of custom shaders that can be dynamically added/removed from the rendering cycle
