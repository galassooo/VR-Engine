Detailed Description

Gestisce la simulazione fisica.

Implementa il pattern Singleton per fornire un accesso centralizzato al sistema fisico. Gestisce collisioni, gravità e altre proprietà fisiche degli oggetti nella scena. Nasconde i dettagli dell'implementazione di Bullet Physics.
