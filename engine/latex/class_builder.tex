\doxysubsection{Detailed Description}
Singleton builder for assembling mesh data and creating \doxylink{class_mesh}{Mesh} objects. 

Accumulates vertex and index data, material, name, and transformation matrix through a fluent interface. After configuration, call build() to generate a \doxylink{class_mesh}{Mesh} and upload its data to the GPU.
