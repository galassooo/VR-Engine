\doxysubsection{Detailed Description}
Core engine class implementing the singleton pattern. 

The \doxylink{class_eng_1_1_base}{Base} class is the main entry point of the graphics engine and manages\+:
\begin{DoxyItemize}
\item Engine initialization and cleanup
\item Scene graph management
\item \doxylink{class_eng_1_1_camera}{Camera} system
\item Rendering pipeline
\item Window management
\end{DoxyItemize}

As a singleton, only one instance can exist at any time, accessed via \doxylink{class_eng_1_1_base_a3c21bcd7bace09d5913ced6de5b15518}{get\+Instance()}.
