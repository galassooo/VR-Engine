\doxysubsection{Detailed Description}
Implements a post-\/processing bloom effect to simulate glow around bright areas. 

This class handles the main steps of the bloom effect\+:
\begin{DoxyEnumerate}
\item Extracts bright areas from the scene
\item Applies a Gaussian blur to these areas
\item Combines the blurred brightness with the original scene 
\end{DoxyEnumerate}
