\doxysubsection{Detailed Description}
Singleton builder for assembling mesh data and creating \doxylink{class_eng_1_1_mesh}{Mesh} objects. 

Accumulates vertex and index data, material, name, and transformation matrix through a fluent interface. After configuration, call \doxylink{class_eng_1_1_builder_aa9f05b6cd9abb5b760f4bb3295223ff3}{build()} to generate a \doxylink{class_eng_1_1_mesh}{Mesh} and upload its data to the GPU.
