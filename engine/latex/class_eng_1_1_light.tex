\doxysubsection{Detailed Description}
Represents a base class for lights in the scene graph. 

The {\ttfamily \doxylink{class_eng_1_1_light}{Light}} class defines the common properties of lights, such as color and intensity, and integrates with the scene graph by inheriting from {\ttfamily \doxylink{class_eng_1_1_node}{Node}}.

Renders the light by uploading its ambient, diffuse, and specular properties to the shader and configuring any subclass-\/specific settings. 

Uses the \doxylink{class_eng_1_1_shader_manager}{Shader\+Manager} to set common material-\/like properties, then calls configure\+Light() with the inverse of the head node\textquotesingle{}s local matrix to place the light in world space.
