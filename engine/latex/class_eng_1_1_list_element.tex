\doxysubsection{Detailed Description}
Represents an element in the render list. 

A \doxylink{class_eng_1_1_list_element}{List\+Element} stores a reference to a scene graph node, its transformation matrix in world coordinates, and its associated material properties. It is used to organize and render nodes in the scene.
