\doxysubsection{Detailed Description}
Represents a light source that emits light in all directions from a single point. 

The {\ttfamily \doxylink{class_eng_1_1_point_light}{Point\+Light}} simulates a localized light source, such as a light bulb, with intensity diminishing over distance based on an attenuation factor. It inherits common light properties from the \doxylink{class_eng_1_1_light}{Light} base class.
