\doxysubsection{Detailed Description}
Renders a cubemap background and provides ambient scene color. 

The \doxylink{class_eng_1_1_skybox}{Skybox} class loads six textures into an Open\+GL cubemap, sets up a cube mesh (VAO/\+VBO), and uses a specialized shader program to draw an infinitely distant environment. It also calculates a weighted average color from the cubemap faces to supply a global ambient color for scene lighting.
