\doxysubsection{Detailed Description}
Represents a base class for lights in the scene graph. 

The {\ttfamily \doxylink{class_light}{Light}} class defines the common properties of lights, such as color and intensity, and integrates with the scene graph by inheriting from {\ttfamily \doxylink{class_node}{Node}}.
