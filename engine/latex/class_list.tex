\doxysubsection{Detailed Description}
A render list that extends \doxylink{class_object}{Object} to integrate with the scene graph. 

The \doxylink{class_list}{List} class stores a collection of nodes in the correct rendering order\+: lights are pushed to the front, and other nodes are pushed to the back.
