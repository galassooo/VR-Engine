\doxysubsection{Detailed Description}
Represents the material properties of a 3D object, including color, shininess, and optional textures. 

The {\ttfamily \doxylink{class_material}{Material}} class encapsulates visual properties used in rendering, such as color, shininess for specular highlights, and an optional diffuse texture.
