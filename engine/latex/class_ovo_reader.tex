\doxysubsection{Detailed Description}
A class responsible for parsing and loading 3D scene data from OVO format files. 

The \doxylink{class_ovo_reader}{Ovo\+Reader} class provides functionality to read and parse OVO format files, constructing a scene graph with meshes, materials, lights, and other 3D scene elements. It maintains internal state about the scene hierarchy and manages the relationships between different scene elements during the parsing process.
