\doxysubsection{Detailed Description}
Abstract base class representing a GPU shader. 

\doxylink{class_shader}{Shader} provides a common interface for creating, loading, and rendering GPU shader programs. Derived classes must implement the create() method to generate the underlying Open\+GL shader object.
